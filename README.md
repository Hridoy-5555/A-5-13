1.জাভাস্ক্রিপ্টে var, let, এবং const ভেরিয়েবল ঘোষণার জন্য ব্যবহৃত হয়। var হলো পুরনো (ES5) পদ্ধতি যা ফাংশন-স্কোপড, রি-ডিক্লেয়ার করা যায়। let (ES6) ব্লক-স্কোপড এবং ভ্যালু আপডেট করা যায় কিন্তু পুনরায় ডিক্লেয়ার করা যায় না। const (ES6) ব্লক-স্কোপড, যা একবার ঘোষণা করলে তার মান পরিবর্তন (reassign) করা যায় না [১, ৪, ৭, ১০]।
